# Contributing to CogniFlow AI

First off, thank you for considering contributing to CogniFlow AI! Your help is invaluable in making this project better. Every contribution is appreciated, whether it's a bug report, a new feature, or an improvement to the documentation.

To ensure a smooth and collaborative process, please review these guidelines.

## Code of Conduct

This project and everyone participating in it is governed by our [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code. Please report unacceptable behavior.

## How Can I Contribute?

### **Reporting Bugs**
- **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/chirag127/CogniFlow-AI-PDF-Conversational-Summarizer-Web-App/issues).
- If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/chirag127/CogniFlow-AI-PDF-Conversational-Summarizer-Web-App/issues/new). Be sure to include a **title and clear description**, as much relevant information as possible, and a **code sample** or an **executable test case** demonstrating the expected behavior that is not occurring.
- Use the **Bug Report** template to ensure all necessary details are included.

### **Suggesting Enhancements**
- Open a new issue and use the **Feature Request** template.
- Clearly describe the proposed enhancement, why it's needed, and provide examples of how it would work.

### **Pull Requests**
1.  **Fork the repository** and create your branch from `main`.
2.  **Set up your development environment** (`npm install`).
3.  **Make your changes** in a separate branch.
4.  **Ensure your code lints** (`npm run lint`).
5.  **Write tests** for any new functionality.
6.  **Ensure all tests pass** (`npm run test`).
7.  **Follow Conventional Commits** for your commit messages (`feat:`, `fix:`, `docs:`, etc.).
8.  **Open a pull request** and link it to any relevant issues. Provide a clear description of the changes.

## Development Setup

1.  **Fork** the repository.
2.  Clone your fork: `git clone https://github.com/YOUR_USERNAME/CogniFlow-AI-PDF-Conversational-Summarizer-Web-App.git`
3.  Install dependencies: `npm install`
4.  Create a `.env` file in the root and add your `CEREBRAS_API_KEY`.
5.  Start the development server: `npm run dev`

## Style Guide

- We use **Biome** for formatting and linting. The rules are defined in `biome.json`.
- Adhere to the **SOLID** principles and general clean code practices.
- Write **self-documenting code**. Comments should explain the "why," not the "what."

Thank you for your contribution!
